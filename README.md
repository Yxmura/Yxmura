Yamura

Building AI systems and Minecraft infrastructure. Learning Rust and cybersecurity.

## Projects

- **Platcha** — Minecraft marketplace with server infrastructure and commerce stack (private, production)
- **Renderdragon** — Tools and resources for Minecraft content creators ([repo](https://github.com/Yxmura/renderdragon))
- **brailk** — VWC (Flemish mathematical notation) to Typst converter ([repo](https://github.com/Yxmura/brailk) / [vwc.yamura.dev](https://vwc.yamura.dev))
- **Packscaler** — AI-powered Minecraft texture pack upscaler using Python ([repo](https://github.com/Yxmura/Packscaler))
- **Velour** — AI companion with persistent memory, preference enforcement, and encrypted sessions ([velourai.me](https://velourai.me))
